# 🌸 ROSE Music Bot (with MongoDB)
A powerful Telegram group music bot.
- YouTube audio/video streaming
- Custom thumbnail & welcome image
- Admin commands, owner broadcast
- Logger & heartbeat
- Persistent user/group data via MongoDB

### Setup
1. Fill `.env` with your credentials
2. Install requirements: `pip install -r requirements.txt`
3. Run: `python main.py`

Deploy to Railway with same env vars.
